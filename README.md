# Quant Research Projects

Independent and co-authored quantitative finance research covering machine learning crash
prediction, event studies, panel econometrics, and time-series volatility modeling. Built
using Python (Pandas, NumPy, scikit-learn, XGBoost) and R (fixest, rugarch).

---

## [Network-Based Early Warning System for Equity Market Crashes](./Network-Crash-Early-Warning)
Composite Fragility Score from network biomarkers across 10 S&P 500 sectors, lifting
crash-prediction AUC from 0.645 to 0.726 using XGBoost. *(Co-authored, under journal review)*

## [Bank Failure Contagion — Event Study (Fama-French Three Factor)](./Bank-Failure-Contagion)
Event study on 37 U.S. bank failures (2015–2025), finding significant negative abnormal
returns (t=-3.00***) in geographically proximate banks. *(Co-authored)*

## [Firm Investment & Volatility Modeling — U.S. Energy Sector](./Energy-Sector-Capex-Volatility)
Panel fixed-effects regression on capex drivers (247 firms, R²=94%) plus ARFIMA-GARCH/APARCH
volatility modeling on ExxonMobil returns. *(Co-authored)*
