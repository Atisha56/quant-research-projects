# Fake News Classification (NLP)

- **Status:** Complete
- **Tools:** Python (scikit-learn, NLTK)

## Overview
Built a Naive Bayes classifier predicting fake vs. real news from headline text alone.

## Key Findings
Achieved 92.1% accuracy on a held-out test set across 32,470 articles, using headline
text as the sole input feature.
